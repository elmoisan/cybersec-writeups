# France Cybersecurity Challenge (FCSC) 2026

## General Information

- **Dates** : April 3, 2026 - April 12, 2026
- **Type** : Jeopardy CTF
- **Organizer** : ANSSI (Agence Nationale de la Sécurité des Systèmes d'Information)

## CTF Format

The FCSC is a **Jeopardy-style** CTF with challenges of varying difficulty levels across the following categories:

- **Crypto** : Cryptanalysis and cryptographic problems
- **Reverse** : Binary reverse engineering
- **Pwn** : Exploit development and application security
- **Web** : Web vulnerabilities and web application security
- **Hardware** : Hardware and embedded systems security
- **Forensics** : Digital forensics and data recovery
- **Side-Channel & Fault Attacks** : Side-channel attacks, fault injection
- **Misc** : Miscellaneous challenges of various nature
